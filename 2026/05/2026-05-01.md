## Upcoming deprecation of GPT-5.2 and GPT-5.2-Codex

**URL:** https://github.blog/changelog/2026-05-01-upcoming-deprecation-of-gpt-5-2-and-gpt-5-2-codex

### 要約
GitHub Copilot の複数の体験で、GPT-5.2 と GPT-5.2-Codex が 2026-06-01 に非推奨になります。代替として GPT-5.5 と GPT-5.3-Codex が案内されており、Copilot Code Review の GPT-5.2-Codex は例外です。

### ユーザーにとって良いこと
利用可能なモデルを前もって把握できるため、Copilot の動作変化を計画的に吸収できます。Enterprise 管理者はポリシー更新を先に済ませられ、利用者側の中断を減らせます。

### アクションプラン
- 6/1 までに Copilot Chat、inline edits、ask、agent mode、code completions で使っているモデル指定を確認し、GPT-5.5 または GPT-5.3-Codex へ切り替えてください。
- Copilot Enterprise 管理者は、モデルポリシーで代替モデルへのアクセスを有効化してください。
- ⚠️ 依存しているワークフローや統合がある場合は、非推奨日より前に対象モデルを更新してください。
