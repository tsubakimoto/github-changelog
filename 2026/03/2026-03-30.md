## Create issues from Slack with Copilot

**URL:** https://github.blog/changelog/2026-03-30-create-issues-from-slack-with-copilot

### 要約
GitHub の Slack アプリから、@GitHub に自然言語で依頼するだけで GitHub Issues を作成できるようになりました。タイトル、本文、担当者、ラベル、マイルストーンまで構造化して作成でき、親子関係のある sub-issue もまとめて作れます。Slack の会話の流れの中で、Issue 作成をそのまま進められるのが大きな変更点です。

### ユーザーにとって良いこと
Slack で議論している内容をそのまま Issue 化できるため、作業の抜け漏れや転記ミスを減らせます。会話の文脈を保ったまま詳細を詰められるので、チームの合意形成からタスク化までが速くなります。

### アクションプラン
- Slack ワークスペースで GitHub アプリをインストールまたは最新版に更新してください。
- `@GitHub` を使って、対象リポジトリに Issue を作成できるか試してください。
- まずは自然言語の Issue 作成フローをチームに共有し、Slack からの依頼テンプレートを決めてください。
- 必要に応じて `@GitHub settings` で、チャンネルごとの既定リポジトリを設定してください。
- ⚠️ この機能は Copilot 有効化済みアカウントが前提なので、利用対象ユーザーを先に確認してください。
