## Request Copilot code review from GitHub CLI

**URL:** https://github.blog/changelog/2026-03-11-request-copilot-code-review-from-github-cli

### 要約
GitHub CLI から Copilot code review を直接依頼できるようになりました。ブラウザや IDE に移動せず、ターミナル中心のフローのまま AI レビューを呼び出せます。

### ユーザーにとって良いこと
CLI を主軸に開発するチームでも、レビュー依頼のたびにコンテキストスイッチせずに済みます。レビューの初速が上がり、PR 作成前後の品質確認をより早い段階で回せます。

### アクションプラン
- GitHub CLI を最新版へ更新し、Copilot code review 呼び出しコマンドをチームで試す
- PR 作成前のセルフチェックや draft PR のレビュー手順に CLI ベースの Copilot review を組み込む
- 自動適用ではないため、CLI 利用ガイドや alias 集に新コマンドを追加

## Major agentic capabilities improvements in GitHub Copilot for JetBrains IDEs

**URL:** https://github.blog/changelog/2026-03-11-major-agentic-capabilities-improvements-in-github-copilot-for-jetbrains-ides

### 要約
JetBrains IDE 向け GitHub Copilot の agentic capabilities が大きく改善されました。より広いコンテキスト理解や複数ステップの支援が強化され、IDE 上での Copilot の主体的な補助が使いやすくなります。

### ユーザーにとって良いこと
JetBrains 利用者は、より関連性の高い提案や複雑な変更支援を受けやすくなります。IDE を離れずに調査、実装、リファクタリングの流れをつなげやすくなり、生産性向上につながります。

### アクションプラン
- IntelliJ IDEA や PyCharm などの JetBrains IDE と Copilot プラグインを最新版へ更新
- agentic 機能を使う前提で、既存の Copilot 運用ルールやプロンプト例を見直す
- 自動適用に近い改善なので、まずは利用者から提案品質の変化を収集

## Explore a repository using Copilot on the web

**URL:** https://github.blog/changelog/2026-03-11-explore-a-repository-using-copilot-on-the-web

### 要約
GitHub の Web UI 上で Copilot を使ってリポジトリを探索できるようになりました。ブラウザだけで主要ファイルの把握や構成理解、質問への応答を進めやすくなります。

### ユーザーにとって良いこと
新しいリポジトリや未経験のコードベースのオンボーディングが速くなります。ローカル環境を立ち上げる前に概要を掴めるため、レビューや調査の入口が軽くなります。

### アクションプラン
- unfamiliar なリポジトリで Copilot の web 探索機能を試し、オンボーディング手順に組み込めるか確認
- 新規参加者向けガイドに、Web での Copilot repo exploration の使い方を追記
- 自動適用の UI 機能として、まずは設計把握やレビュー前の事前調査で活用
