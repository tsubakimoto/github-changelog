## Copilot auto model selection is generally available in JetBrains IDEs

**URL:** https://github.blog/changelog/2026-03-12-copilot-auto-model-selection-is-generally-available-in-jetbrains-ides

### 要約
JetBrains IDEs で Copilot の auto model selection が一般提供になりました。ユーザーが毎回モデルを選ばなくても、タスクに応じたモデル選択を Copilot に任せられます。

### ユーザーにとって良いこと
モデル選択の認知負荷が下がり、提案の質と操作の手軽さを両立しやすくなります。JetBrains 上での Copilot 利用を標準化しやすい改善です。

### アクションプラン
- JetBrains IDE と Copilot プラグインを最新版へ更新し、auto model selection を試用
- 手動モデル選択を前提にしたチームガイドがある場合は、auto mode との使い分けを整理
- 自動適用に近い機能改善なので、まずは提案品質の変化を観察してフィードバック

## Actions OIDC tokens now support repository custom properties

**URL:** https://github.blog/changelog/2026-03-12-actions-oidc-tokens-now-support-repository-custom-properties

### 要約
GitHub Actions の OIDC トークンで、repository custom properties を claim として扱えるようになりました。リポジトリ固有の属性をクラウド認証や外部システム連携に渡しやすくなります。

### ユーザーにとって良いこと
デプロイ先や権限判定を、リポジトリ属性に応じて柔軟に制御しやすくなります。CI/CD の認可やトレーサビリティを強化したい組織に有用です。

### アクションプラン
- OIDC を使うデプロイ workflow で、custom properties を claim に含める設計が有効か見直す
- クラウド側の trust policy / federation 設定で受け取る claim を拡張できるか確認
- 自動適用ではないため、Actions とクラウド認証設定の両方に変更計画を用意

### リファレンス
- [Actions OIDC tokens now support repository custom properties](https://github.blog/changelog/2026-03-12-actions-oidc-tokens-now-support-repository-custom-properties)

## Issue fields: Structured issue metadata is in public preview

**URL:** https://github.blog/changelog/2026-03-12-issue-fields-structured-issue-metadata-is-in-public-preview

### 要約
Issue に構造化された metadata fields を持たせる機能が public preview として公開されました。Priority や Effort、日付などを label ではなく型付きフィールドで管理できるようになります。

### ユーザーにとって良いこと
Issue 管理の標準化や検索性、レポート性が上がります。Projects や横断検索と組み合わせた運用でも、label 依存の曖昧さを減らせます。

### アクションプラン
- public preview 対象なら、既存 label のうち構造化フィールドへ移せる項目を洗い出す
- Issue template や運用ルールを、typed metadata 前提へ段階的に移行する計画を立てる
- プレビュー機能なので、本番運用前に対象チームを絞って試験導入

### リファレンス
- [Issue fields: Structured issue metadata is in public preview](https://github.blog/changelog/2026-03-12-issue-fields-structured-issue-metadata-is-in-public-preview)
