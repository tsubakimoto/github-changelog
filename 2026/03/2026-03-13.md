## Optionally skip approval for Copilot coding agent Actions workflows

**URL:** https://github.blog/changelog/2026-03-13-optionally-skip-approval-for-copilot-coding-agent-actions-workflows

### 要約
Copilot coding agent が開いた PR や push で起動する GitHub Actions workflow について、repository administrators が人手承認を省略できる設定が追加されました。既定では従来どおり手動承認が必要で、必要なリポジトリだけ即時実行へ切り替えられます。

### ユーザーにとって良いこと
信頼できる環境では、Copilot の変更に対するテストや検証のフィードバックループを短縮できます。一方で、承認を残す既定値により、セキュリティ重視の運用も維持しやすくなっています。

### アクションプラン
- Copilot coding agent を使うリポジトリごとに、Actions の secrets / permissions リスクを評価
- 低リスクなリポジトリだけで skip approval を有効化し、テスト速度の改善を検証
- ⚠️ 明示設定が必要なオプトイン機能なので、セキュリティレビューなしに全リポジトリへ展開しない

## Self-hosted runner minimum version enforcement paused

**URL:** https://github.blog/changelog/2026-03-13-self-hosted-runner-minimum-version-enforcement-paused

### 要約
GitHub Actions の self-hosted runner 最小バージョン要件 `v2.329.0` の enforcement が一時停止されました。現時点では古いバージョンでも registration / configuration を継続できますが、長期的な方向性として古い runner のブロック方針自体は維持されています。

### ユーザーにとって良いこと
大規模環境でも移行猶予を確保しながらアップグレード計画を進められます。急な登録停止による運用混乱を避けつつ、計画的な更新に移れるのが利点です。

### アクションプラン
- self-hosted runner のバージョンを棚卸しし、`v2.329.0` 以上への更新計画を継続
- auto-update 無効 runner や ARC `disableUpdate=true` の構成では手動更新手順を明文化
- ⚠️ enforcement は paused でも、古すぎる runner が workflow 実行を拒否される通常の deprecation process は継続するため放置しない
- GHES は今回の変更の影響対象外なので、GitHub.com / Enterprise Cloud と分けて運用判断する

## Updates to GitHub Copilot for students

**URL:** https://github.blog/changelog/2026-03-13-updates-to-github-copilot-for-students

### 要約
GitHub Education benefits を持つ学生向けに、新しい GitHub Copilot Student plan への移行が始まりました。モデル提供内容が更新され、学生向けに長期的で持続可能な Copilot 体験へ再編されています。

### ユーザーにとって良いこと
学生は引き続き Copilot を学習用途で活用しやすくなります。プラン整理により、教育向け AI 学習支援の継続性が高まる点が大きなメリットです。

### アクションプラン
- 学生ユーザーは plan 移行後の利用条件やモデル提供内容を Community Discussion で確認
- 教育機関やメンターは、受講者向けガイドを新しい Student plan 前提に更新
- 自動移行ベースの更新として、まずは学生アカウントで利用可能機能の変化を周知

- [Updates to GitHub Copilot for students](https://github.blog/changelog/2026-03-13-updates-to-github-copilot-for-students)
