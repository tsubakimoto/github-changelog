## Secret scanning pattern updates — March 2026

**URL:** https://github.blog/changelog/2026-03-10-secret-scanning-pattern-updates-march-2026

### 要約
GitHub の secret scanning パターンが 2026 年 3 月版へ更新され、より多くの認証情報や機密値を検出できるようになりました。既存の secret scanning 運用でも、検知範囲の拡張によって流出検知の精度向上が期待できます。

### ユーザーにとって良いこと
誤ってコミットされたシークレットを早期に見つけやすくなり、資格情報漏えいやクラウド侵害のリスクを下げられます。継続的に secret scanning を使っている組織ほど恩恵を受けやすい更新です。

### アクションプラン
- secret scanning を有効化済みのリポジトリでは、新しい検知対象が運用にどう影響するかをセキュリティチームへ共有
- 未導入のリポジトリでは、優先度の高いコードベースから secret scanning の有効化を検討
- 自動適用の検知範囲拡張として、アラート対応手順や誤検知のトリアージ基準を見直す

## CodeQL 2.24.3 adds Java 26 support and other improvements

**URL:** https://github.blog/changelog/2026-03-10-codeql-2-24-3-adds-java-26-support-and-other-improvements

### 要約
CodeQL 2.24.3 が公開され、Java 26 のサポートと各種改善が追加されました。最新の Java バージョンを使うプロジェクトでも CodeQL による静的解析を継続しやすくなります。

### ユーザーにとって良いこと
新しい言語機能や構文を使うコードでも、セキュリティ解析と品質チェックを保ちやすくなります。セキュリティツールが言語アップデートに追従することで、開発速度と安全性を両立しやすくなります。

### アクションプラン
- CodeQL を使っている CI/CD では、解析バージョンを 2.24.3 に更新できるか確認
- Java 26 を使うリポジトリでは、スキャン結果とビルド互換性を staging で検証
- 自動更新に頼らず pin している場合は、依存更新計画に CodeQL 2.24.3 を追加

## Dependabot now supports pre-commit hooks

**URL:** https://github.blog/changelog/2026-03-10-dependabot-now-supports-pre-commit-hooks

### 要約
Dependabot が `pre-commit` フレームワークで使われる hooks の更新に対応しました。`pre-commit` 設定のメンテナンスを手動で追いかけなくても、Dependabot 経由で更新提案を受けられます。

### ユーザーにとって良いこと
コード品質やセキュリティに関わる hooks を古いまま放置しにくくなります。日常的なメンテナンス負荷を減らしながら、開発体験と安全性を維持しやすくなります。

### アクションプラン
- `pre-commit` を利用しているリポジトリで Dependabot 設定を見直し、更新提案が出ることを確認
- 生成された PR を通常の依存更新と同様にレビュー・マージする運用へ統合
- opt-in 的な設定変更が必要な場合は `.github/dependabot.yml` の更新を計画

### リファレンス
- [Dependabot now supports pre-commit hooks](https://github.blog/changelog/2026-03-10-dependabot-now-supports-pre-commit-hooks)
