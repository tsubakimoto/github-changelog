## GitHub Copilot CLI now supports Copilot auto model selection

**URL:** https://github.blog/changelog/2026-04-17-github-copilot-cli-now-supports-copilot-auto-model-selection

### 要約
Copilot auto model selection が GitHub Copilot CLI で一般提供になり、すべての Copilot プランで利用できるようになりました。Auto はその時点で最も効率的なモデルを選び、GPT-5.4、GPT-5.3-Codex、Sonnet 4.6、Haiku 4.5 などに動的にルーティングします。CLI 上では実際に使われたモデルを確認でき、必要ならいつでも特定モデルに切り替えられます。

### ユーザーにとって良いこと
モデル選択の手間が減り、レート制限を避けながら安定して Copilot CLI を使いやすくなります。管理者ポリシーを尊重したまま、自動で最適化されたモデル利用ができます。

### アクションプラン
- Copilot CLI で `auto` モードを試し、既存の固定モデル指定と比べて応答速度と品質を確認してください。
- チームの利用ガイドに、必要に応じて特定モデルへ切り替える手順と、CLI で実際の使用モデルを確認する方法を追記してください。
- ⚠️ Auto の請求は選択されたモデル倍率に連動します。利用量の多い環境は、0x〜1x モデル中心でも premium request 消費を確認してください。
