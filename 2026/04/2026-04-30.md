## GitHub Copilot in Visual Studio — April update

**URL:** https://github.blog/changelog/2026-04-30-github-copilot-in-visual-studio-april-update

### 要約
Visual Studio 向け GitHub Copilot の 4 月更新では、IDE から直接クラウドエージェントを起動できるようになり、よりエージェント主導の作業フローが強化されました。あわせて、ユーザー単位のカスタムエージェント、追加の skills 検出場所、新しい Debugger agent、チャット履歴パネルなど、日常的な開発作業を支える機能が広く追加されています。

### ユーザーにとって良いこと
Visual Studio の中だけで、調査・修正・検証までの流れを Copilot に任せやすくなります。個人用カスタムエージェントやショートカットのカスタマイズも増え、チームと個人の両方で使い勝手が上がります。

### アクションプラン
- Visual Studio Insiders でクラウドエージェント起動フローを試し、issue 生成から PR 作成までの流れを確認してください。
- `.github/skills/` に加えて `.claude/skills/` と `.agents/skills/` も含めた skills 配置方針を整理し、チーム標準を決めてください。
- `Edit.AcceptSuggestion` 系のショートカットを見直し、Copilot の受け入れ操作を自分のキー配置に合わせてください。
- ⚠️ Debugger agent は実行時検証を伴うため、導入前に対象リポジトリと権限範囲を確認してください。
