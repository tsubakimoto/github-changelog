## Copilot Student GPT-5.3-Codex removal from model picker

**URL:** https://github.blog/changelog/2026-04-27-copilot-student-gpt-5-3-codex-removal-from-model-picker

### 要約
Copilot Student plan では、モデルピッカーから GPT-5.3-Codex が削除されました。今後も auto model selection では利用でき、各リクエストに最適なモデルを自動で選ぶ運用に寄せる変更です。あわせて、この更新は Copilot Individual / Student 向けの一時的な reliability and performance measures の一部として案内されています。

### ユーザーにとって良いこと
モデル選択を迷う場面が減り、学生向けプランをより軽く保ちやすくなります。auto model selection に寄せることで、設定を切り替える手間を減らしながら、必要な性能を確保しやすくなります。

### アクションプラン
- 学生向けの案内や手順書に GPT-5.3-Codex の直接選択が残っていないか確認してください。
- Copilot Student を使うチームは、明示的なモデル選択ではなく auto model selection 前提の運用へ切り替えてください。
- ⚠️ 影響は Copilot Student plan に限定されます。モデルの直接選択は消えますが、auto selection では引き続き利用できます。

- [Copilot Student GPT-5.3-Codex removal from model picker](https://github.blog/changelog/2026-04-27-copilot-student-gpt-5-3-codex-removal-from-model-picker)

## Copilot cloud agent starts 20% faster with Actions custom images

**URL:** https://github.blog/changelog/2026-04-27-copilot-cloud-agent-starts-20-faster-with-actions-custom-images

### 要約
Copilot cloud agent の起動が、GitHub Actions custom images を使った runner 環境の最適化により 20% 以上高速化されました。issue の割り当て、Agents タブからの task 開始、PR での @copilot メンションなど、cloud agent が起動する入口全体に効く改善です。3 月に出た 50% の起動改善に続く追加の短縮です。

### ユーザーにとって良いこと
Copilot が作業を始めるまでの待ち時間が短くなり、レビュー依頼やタスク着手のフィードバックループが速くなります。日常的に agent を使うほど、体感の改善が大きくなります。

### アクションプラン
- Cloud agent を使うチームには、起動時間がさらに短くなったことを周知してください。
- 既存の Copilot cloud agent ワークフローはそのままで恩恵を受けられるため、追加設定が不要かを確認してください。
- ⚠️ この改善は cloud agent の runner 環境に適用される変更です。自前の Actions image 設定を持つ場合は、Copilot の動作前提と合わせて確認してください。

## GitHub Copilot code review will start consuming GitHub Actions minutes on June 1, 2026

**URL:** https://github.blog/changelog/2026-04-27-github-copilot-code-review-will-start-consuming-github-actions-minutes-on-june-1-2026

### 要約
2026-06-01 から、Copilot code review は AI Credits に加えて GitHub Actions minutes も消費するようになります。対象は private repositories 上で実行される review で、public repositories は引き続き無料です。GitHub-hosted runners を使う前提のため、既存の Actions 利用量と予算への影響を早めに見積もる必要があります。

### ユーザーにとって良いこと
請求の仕組みが事前に明示されたので、導入済みのチームは予算調整や利用方針の見直しを計画しやすくなります。public repositories には影響がなく、使い方に応じたコスト管理ができます。

### アクションプラン
- 6 月 1 日までに、現在の GitHub Actions minutes の消費量と残量を確認してください。
- Billing managers と engineering leads に共有し、Copilot code review の利用増加が予算を圧迫しないか見直してください。
- 予算管理が必要な組織は、GitHub Actions の spending limits と budgets を調整してください。
- ⚠️ private repositories での review は 2026-06-01 以降、Actions minutes の対象になります。無料の public repositories とは扱いが異なります。
