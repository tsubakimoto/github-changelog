## Copilot usage metrics now identify active and passive Copilot code review users

**URL:** https://github.blog/changelog/2026-04-06-copilot-usage-metrics-now-identify-active-and-passive-copilot-code-review-users

### 要約
Copilot の利用指標で、Copilot Code Review の利用を「active」と「passive」に分けて追跡できるようになりました。Enterprise/organization 管理者は、日次および 28 日集計のユーザーレポートで、実際に操作した利用と自動実行された利用を区別できます。

### ユーザーにとって良いこと
単なるカバレッジではなく、実際の関与度を把握できます。Copilot Code Review がどれだけ適用されているかと、ユーザーがどれだけ積極的に使っているかを分けて見られるため、導入状況の評価がしやすくなります。

### アクションプラン
- API 連携やダッシュボードで `used_copilot_code_review_active` と `used_copilot_code_review_passive` を扱うように更新してください。
- Copilot の導入レポートを、カバレッジ指標とエンゲージメント指標の両方で見直してください。
- `used_agent` や `used_copilot_coding_agent` と並べて、Copilot 各機能の利用状況を比較できる指標設計にしてください。
